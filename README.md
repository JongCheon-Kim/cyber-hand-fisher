🎮 CYBER HAND FISHER (사이버 핸드 피셔)

이 게임은 사이버네틱 무드의 심해 속에서 비선형 곡선 궤도로 날아가는 AI 집게발을 제어하여 수산물을 수확하고, 가공 납품과 미식 요리 레시피 개발까지 한 번에 즐기는 하이브리드 아케이드 게임기입니다.

화려한 네온 비주얼과 비선형 물리 궤도의 짜릿한 손맛, 그리고 AI 요리 연구소의 두뇌 플레이가 융합된 독창적인 게임 시스템을 경험해 보세요!

==================================================

[🕹️ 게임 플레이 방법]

본 게임기는 타겟 조준 록온, AI 집게 격발, 가공품 납품, AI 레시피 개발의 4단계 루프로 진행됩니다.

어종 조준 (Target Select Matrix)

목표 설정: 화면 하단의 어종 선택 매트릭스에서 사냥할 수산물(고등어, 연어, 대구, 멍게, 오징어 등)을 클릭하여 타겟으로 지정합니다.

록온 시스템: 타겟이 선택되면 해수 속 해당 어종에 조준 마커가 활성화되며, AI 집게발의 유도 대상이 됩니다.

비선형 집게 격발 (Launch AI Auto-Harvest!)

자동 추격 격발: 타겟 지정 완료 후 하단의 주황색 Launch AI Auto-Harvest! 버튼을 누릅니다.

짜릿한 궤적: 화면 좌측 상단에 대기 중이던 메카니컬 AI 집게가 비선형 곡선 궤도를 그리며 수중의 물고기를 가로채기 위해 발사됩니다.

공정 제어 및 납품 (Physical AI Process Control)

가공 및 의뢰 해결: 우측 상단의 피지컬 AI 공정 제어반과 실시간 납품 의뢰판을 상시 확인하세요.

미션 수행: 예컨대 '멍게 [HACCP 멸균 통조림] (0/1)'과 같은 납품 의뢰가 들어왔을 때, 해당 어종을 수집 보관함에 적치해 조리 규격에 맞춰 납품합니다.

하이스코어: 의뢰를 완수하면 좌측 하단의 스코어 보드에 실시간 점수(SCORE)와 포인트(PTS)가 쌓입니다.

AI 미식 요리 연구소 (AI Gourmet Lab)

융합 연금술: 게임 중 수확하여 보관함에 담긴 수산물 중 어울리는 두 가지 재료를 선택하여 연구소 슬롯에 등록합니다.

레시피 개발: 우측 하단의 레시피 개발 버튼을 누르면, 연동된 Gemini AI 엔진이 두 재료가 결합한 가상의 독창적인 미식 요리 레시피를 동적으로 탄생시킵니다.

AI 기능 활성화: 우측 최하단의 Gemini API Key 입력란에 본인의 API Key를 입력하면 요리 연구소의 AI 레시피 생성 기능이 완전히 개방되어 모든 콘텐츠를 즐길 수 있습니다.
